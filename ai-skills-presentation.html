The Basics

What Is an AI Skill?

Think of skills as apps for AI agents. Just like your phone becomes more useful with each app you install, AI agents become more capable with each skill they learn.

A skill is a reusable package of instructions, tools, and context that an AI agent can load on demand to perform a specific task.

A Skill Typically Contains

📋
Instructions
What to do and how to do it
🔧
Tools & Scripts
Executable capabilities
📚
Reference Data
Templates, styles, standards
🎯
Trigger Logic
When to activate automatically

The Intellectual Roots

Where It All Started

MAY 2022

MRKL Systems

AI21 Labs proposed Modular Reasoning, Knowledge and Language — the first architecture for LLMs that could call external tools and modules. The seed of the "tool-augmented AI" idea.

OCT 2022

ReAct

Princeton researchers formalized the Reason + Act paradigm — LLMs that think step-by-step, then take action. This became the blueprint for every modern AI agent.

The Breakthrough Paper

Voyager: The First "Skill Library"

In May 2023, researchers from NVIDIA, Caltech, UT Austin, and Stanford introduced Voyager — an AI agent that could learn, store, and reuse skills.

Skills compound over time. An agent with a growing skill library gets better at solving new problems — without retraining.

— Voyager paper, arXiv:2305.16291

Why Voyager Mattered

Reusable code
Skills stored as functions, retrieved by similarity
Compositional
Complex tasks built from simpler skills
Ever-growing
Library expands with each new challenge
No forgetting
Past capabilities preserved permanently

The Industry Response

From Plugins to Function Calling

Mar 2023
ChatGPT Plugins launch — the first mainstream attempt at "skills for chat." Third-party services could integrate directly with ChatGPT.
Jun 2023
OpenAI ships Function Calling — a structured, schema-based interface replacing free-form plugin calls. Models learn to output structured JSON to invoke tools.
Nov 2023
Custom GPTs replace plugins — OpenAI pivots from developer-only plugins to user-creatable agents. Anyone can bundle instructions + tools + knowledge into a "GPT."
Apr 2024
Plugins officially retired — the experiment ends, but the pattern survives. The industry learns: skills need structure, not just APIs.

The Turning Point

MCP: The Universal Connector

In November 2024, Anthropic released the Model Context Protocol — an open standard for connecting AI agents to external tools and data.

Think of MCP as USB for AI: one standard interface that works across all models and all tools.

Open Standard JSON-RPC Protocol Linux Foundation

Adopted By

Anthropic · OpenAI · Google DeepMind · Microsoft · 16,000+ community servers

Why It Won

Solved the M×N problem. Instead of building separate integrations for every model-tool pair, build one MCP server and it works everywhere.

MCP ≠ Skills — MCP is the connectivity protocol (how agents talk to tools). Skills are packaged knowledge (what agents know how to do). They're complementary: MCP connects, Skills instruct.

Under The Hood

How Skills Work Today

Modern AI skills follow a simple, portable format: a folder with a manifest file and supporting resources.

// A typical skill folder
my-skill/
├── SKILL.md        // Instructions
├── templates/     // Reference data
├── scripts/       // Executable tools
└── examples/      // Usage patterns

// The agent loads the skill
// when it matches the task
trigger: "user mentions PDF"

Your Takeaway

What This Means for You

As a leader, skills change how you should think about AI adoption.

Action Items

1. Start with pre-built skills

Most platforms ship with skills for documents, code review, and data analysis. Use them before building custom.

2. Identify repeatable workflows

If your team does the same 5-step process weekly, it's a skill candidate. Capture institutional knowledge.

3. Think ecosystems, not tools

Choose platforms that support open standards (MCP). Avoid vendor lock-in on proprietary skill formats.

4. Empower skill creators

Your domain experts can create skills without ML knowledge. Encourage experimentation.
